# British Airways – Customer Insights & Data Analytics Project

##  Overview
This project analyzes **British Airways’ customer data** to identify travel patterns, loyalty insights, and opportunities for service improvement.  
A **lookup table system** and **data visualization dashboard** were developed to automate lounge eligibility and support decision-making.

##  Key Tasks
- Created a dynamic **Excel lookup table** to automate **lounge eligibility checks**.  
- Conducted **Exploratory Data Analysis (EDA)** to uncover customer trends and satisfaction patterns.  
- Developed a **PowerPoint presentation** summarizing findings and recommendations for business enhancement.  
- .

##  Tools Used
**Excel**, **Data Visualization**, **EDA**

##  Outcome
Delivered actionable insights on customer behavior and streamlined the lounge eligibility process through automated lookup systems.
